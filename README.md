# Lego组件化开发初体验


## 目录结构

```
- vue-lego/
  + package.json //npm配置文件
  + webpack.config.js //webpack配置
  + index.html //页面
  - node_modules //npm加载的模块
  - src //开发资源目录
    - assets //一些资源
    - components //vue组件
```


## 打包运行查看

* 打包：

```
npm run build
```

* 运行

```
npm run dev
```

* 查看

```
浏览器中访问 localhost:8080
```
